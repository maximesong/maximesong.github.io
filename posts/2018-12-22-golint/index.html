使用Golang Lint

Sat, Dec 22, 2018

Go语言本身有非常不错的静态检测工具，比如golint，govet。但是如果有特别多的额外检测工具需要使用，可能效率就会变低。可以使用工具达到这个目的golangci-lint。

最简单的方式是以默认方式运行:

golangci-lint run

有时可能希望对一些包做过滤，可以考虑使用命令

go list ./... | golangci-lint run

如果希望忽略测试，可以加上--tests=false

go list ./... | golangci-lint run --tests=false
